## 概要
scanner-appは、撮影した書類画像から歪みや影を除去し、スキャナーで取り込んだような画像へ補正するPythonアプリケーションです。

OpenCVを用いた画像処理技術を活用し・書類の検出・透過変換・影の除去・二値化を行います。


## 開発目的
Pythonを用いたアプリ開発の経験を得るために作成しました。

小学校から大学・専門学校まで、書類をスマートフォンで撮影し保存・提出する機会は多くあります。

しかし実際に撮影した画像には
・斜めからの撮影
・影が映り込む
・書類の輪郭がゆがむ
といった問題が多く、視認性が低下します。

そこで、誰でも簡単にスキャン画像のような品質へ補正できるアプリケーションの開発を目指しました。


## 主な機能
書類領域の検出
輪郭抽出
透視変換
グレースケール変換
ノイズ除去
適応性二値化によるスキャン画像生成
